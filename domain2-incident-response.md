# Incident Response

## Security Incident
A security incident is any event that threatens the confidentiality,
integrity, or availability of systems or data.

## Incident Response Phases
1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

## Importance of Logging
Logs help security teams detect incidents, investigate events,
and understand what happened during an attack.
